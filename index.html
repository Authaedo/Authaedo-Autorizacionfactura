Prototipo de tablero interno. Los datos de las facturas se guardan de forma compartida: cualquiera que abra este mismo tablero verá las mismas facturas y estados. Los mails se envían con EmailJS: completá SERVICE_ID, TEMPLATE_ID y PUBLIC_KEY al inicio del script (buscá "CONFIGURACIÓN EMAILJS") con los datos de tu cuenta, y armá en EmailJS un template con las variables to_email, to_name, asunto y mensaje. El PDF cargado queda adjunto a la factura para poder abrirlo o descargarlo en cualquier momento, incluida una inspección.
